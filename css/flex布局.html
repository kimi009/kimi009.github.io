flex布局 #

flex布局详情

flex属性值解释 、默认值：flex: 0 1 auto; #

①flex-grow：设置弹性盒子的伸展占宽/比例。默认为0，即不会进行放大。

②flex-shrink：设置弹性盒子的缩小占宽/比例。默认为1

③flex-basis：设置弹性盒子的伸缩值。默认为auto

demo1 #

<div style="display: flex;margin: auto;width: 400px;height: 6rem;border: 0.1rem solid black;">
	<div style="flex: 3 0 200px; background: rgb(211, 118, 209)">我是aa</div>
	<div style="flex: 2 0 100px; background: rgb(13, 189, 89)">我是bb</div>
</div>

demo2 #

<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8" />
		<meta http-equiv="X-UA-Compatible" content="IE=edge" />
		<meta name="viewport" content="width=device-width, initial-scale=1.0" />
		<title>Document</title>
		<style>
			.container {
				display: flex; /* 表示这个盒子使用Flex弹性布局 */
				margin: auto; /* 表示使这个盒子在页面上垂直居中*/
				width: 400px;
				height: 6rem;
				border: 0.1rem solid black;
				background: rgb(169, 166, 166);
			}
			.aa {
				width: 100px;
				flex-grow: 1;
				flex-shrink: 1;
				flex-basis: auto;
				background: rgb(247, 160, 174);
			}
			.bb {
				width: 100px;
				flex: 3 1 50px;
				background: rgb(163, 160, 247);
			}
		</style>
	</head>
	<body>
		<div class="container">
			<div class="aa">我是盒子aa</div>
			<div class="bb">我是盒子bb</div>
		</div>
	</body>
</html>

一个盒子的真正尺寸=flex-basis属性值+flex-grow属性值。

总结

①在width、flex-basis同时存在的情况下，flex-basis > width，width无作用。
	就像bb一样，同时存在width和flex-basis,此时计算宽度是按照flexbasis来计算的也就是50

②在不具备空间可用于盒子占宽/扩展的情况下，flex-grow无作用。(就像上面的如果aa是1 1 100， bb是1 1 300)，这样加起来跟总宽度400一样了，没剩余空间了，此时的flex-grow没用了

③盒子的真实尺寸为flex-basis属性值+flex-grow属性值。

④当盒子的flex-shrink属性值为0时，该盒子不会进行缩小。

<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8" />
		<meta http-equiv="X-UA-Compatible" content="IE=edge" />
		<meta name="viewport" content="width=device-width, initial-scale=1.0" />
		<title>Document</title>
		<style>
			.container {
				display: flex; /* 表示这个盒子使用Flex弹性布局 */
				margin: auto; /* 表示使这个盒子在页面上垂直居中*/
				width: 400px;
				height: 6rem;
				background: rgb(169, 166, 166);
			}
			.aa {
				width: 100px;
				flex:1 1 100px;
				background: rgb(247, 160, 174);
			}
			.bb {
				width: 100px;
				flex: 1 1 60%;
				background: rgb(163, 160, 247);
			}
			.cc {
				width: 100px;
				flex: 1 1 100px;
				background: rgb(123, 60, 47);
			}
		</style>
	</head>
	<body>
		<div class="container">
			<div class="aa">我是盒子aa</div>
			<div class="bb">我是盒子bb</div>
			<div class="cc">我是盒子cc</div>
		</div>
	</body>
</html>

  • 这里一定要注意一个问题，就是怎么计算各个的实际宽度

    • 首先要看bb所占的百分比，目前是60%，flex-basis就是400 * 60% = 240

    • 然后发现aa+bb+cc也就是100+240+100 >400 这时候就要缩放，缩放比例是400/440

    • 因此aa实际的宽度是 100 * 400 /440 = 90.909090 ,四舍五入就是90.91

      注意这个比例的计算方式：400/440，也就是理论宽度400/ 实际换算宽度440

    • cc和aa的flex-basics是一样的，所以都是90.91

    • 最后看bb 目前flex-basis是240，按目前的比例缩放就是240 * 400 / 440 = 218.181818 ，四舍五入就是218.19

    • 以上就是实际的计算方式
